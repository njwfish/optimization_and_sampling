\section{Optimization}
\label{sec:optimization}

Optimization starts from a simple task: given an objective \(f:\R^d\to\R\), choose \(x\) so that \(f(x)\) is small.
The first questions are local.
If we stand at a point \(x\), which direction decreases \(f\), how far should we move, and what extra information is carried by curvature?
This unit develops the gradients, Hessians, curvature bounds, and step-size rules that answer those questions, then extends the same ideas to noisy gradients, structured linear algebra, and feasibility constraints.
Later the same local information will reappear in sampling, augmentation, and diffusion.
